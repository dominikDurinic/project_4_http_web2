HTTP Test

Dobrodošli u aplikaciju za ispitivanje načina rada i performansi HTTP protokola!

Ispitujemo razlike performansa u pogledu brzine prijenosa datoteka verzije protokola HTTP/2 i HTTP/3 u odnosu na starije verzije istoimenog protokola HTTP/1.x aplikacijskog sloja.

O protokolu

HTTP (Hypertext Transfer Protocol) je protokol aplikacijskog sloja u modelu skupa internetskih protokola za distribuirane, suradničke, hipermedijske informacijske sustave.

Osnova funkcioniranja se temelji na modelu zahtjev - odgovor između klijenta i poslužitelja.

Postoje neke zajedničke informacije koje su općenito prisutne u svim HTTP zahtjevima. Oni su navedeni u nastavku.

  • HTTP Version
  • URL
  • HTTP Method
  • HTTP Request Headers
  • HTTP Body

Najčešće korištene HTTP metode su HTTP GET i HTTP POST

Postoje razne stvari sadržane u HTTP odgovoru, neke od njih su navedene u nastavku.

  • HTTP Status Code
  • HTTP Headers
  • HTTP Body

Sljedećih pet zapisa predstavlja analizu statusnih kodova:

  • 1x Informative
  • 2xx Achievement
  • 3xx Reorientation
  • 4xx Client Mistake
  • 5xx Error on the Server

Također bitno je naglasiti da je ovaj protokol "stateless" odnosno ne čuva stanje i zahtjevi su međusobno nezavisni.

Sigurnija inačica HTTP protokola naziva se HTTPS (HyperText Transfer Protocol Secure). Podaci se pri prijenosu kriptiraju i tako sogurnije šalju između klijenta i poslužitelja.
